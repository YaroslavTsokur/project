Таблица Тегов:

ТегЕго функции
<!--...-->Используется для добавления комментариев.
<!DOCTYPE>Объявляет тип документа и предоставляет основную информацию для браузера — его язык и версия.
<a>Гиппертекстовая ссылка
<abbr>Определяет текст как аббревиатуру. Поясняюий текст задается с помощью атрибута <title>.
<address>Задает контактные данные автора/владельца документа или статьи. Отображается в браузере курсивом.
<area>Представляет собой гиперссылку с текстом, соответствующей определенной области на карте-изображении или активную область внутри карты-изображения. Всегда вложен внутрь тега <map>.
<article>Независимая часть документа (статья в журнале, запись в блоге и т.д.)
<aside>Отображает раздел, который имеет косвенное отношение к сайту.
<audio>Звуковой контент.
<b>Полужирный шрифт.
<base>Задает базовый адрес, относительно которого вычисляются относительные адреса.
<bdi>Изолирует отрывок текста, написанный на языке, в котором чтение текста происходит справа налево, от остального текста.
<bdo>Отображает текст в направлении, указанном в атрибуте <dir>, переопределяя текущее направление написания текста.
<blockquote>Выделяет текст, как цитату.
<body>Тело документа (содержимое, не относящееся к метаданным.
<br>Перенос строки.
<button>Интерактивная кнопку
<canvas>Контейнер-холст для простых изображений.
<caption>Подпись к таблице.
<cite>Источник цитирования.
<code>Фрагмент программного кода.
<col>Выбирает для форматирования 1 или несколько столбцов таблицы.
<colgroup>Создает структурную группу столбцов, обозначающих множество однородных ячеек.
<data>Элемент используется для связывания значения атрибута <value>, которое представлено в машиночитаемом формате и может быть обработано компьютером, с содержимым тега.
<datalist>Элемент-контейнер для выпадающего списка элемента <input>
<dd>Описание термина.
<del>Перечеркивает текст.
<details>Интерактивный виджет, который можно открыть и закрыть.
<dfn>Термин, выделенный курсивом.
<dialog>Интерактивный элемент (окно, поисковое окно), с которым взаимодействует пользователь для выполнения задач.
<div>Контейнер для группировки блочных элементов.
<dl>Контейнер с термином и его описанием.
<dt>Используется для задания термина.
<em>Курсив
<embed>Внешний интерактивный контент или плагин.
<fieldset>Рамки вокруг элементов.
<figcaption>Заголовок для тега <figure>.
<figure>Контейнер для иллюстрации, диаграммы, фотографии и т.д.
<footer>Завершающая область, нижний колонтитул.
<form>Форма для собра данных от пользователя.
<h1-h6>Заголовки шести уровней для связанных разделов.
<head>Метаданные страницы.
<header>Вводная информация сайта или группы навигационных ссылок.
<hr>Горизонтальная линия для тематического разделения.
<html>Корневой элемент. Сообщает браузеру, что это HTML-документ.
<i>Курсив без дополнительного акцента.
<iframe>Встроенный фрейм (другой документ).
<img>Вставка изображения.
<input>Многофункциональные поля-формы для ввода данных.
<ins>Подчеркивание.
<kbd>Выделяет текст, который должен быть введен пользователем.
<label>Текстовая метка для элемента <input>.
<legend>Заголовок элементов формы <input>.
<li>Элемент маркированного или нумерованного списка.
<link>Определяет отношения между документом и внешним ресурсом. Также используется для подключения внешних таблиц стилей.
<main>Контейнер для основного содержимого (не более 1 на страницу).
<map>Активные области на карте-изображении.
<mark>Выделение желтым фоном.
<meta>Дополнительная информация о сайте.
<meter>Индикатор измерения в заданном диапазоне.
<nav>Навигационные ссылки по сайту.
<noscript>Секция, не поддерживающая скрипты.
<object>Контейнер для встраивания мультимедиа.
<ol>Упорядоченная нумерация (числовая или алфавитная).
<output>Поля для вывода результата вычисления, рассчитанного с помощью скрипта.
<param>Параметры для плагинов, встраиваемых с помощью <object>.
<picture>Дает возможность браузеру выбрать наилучшее изображение (одно изображения, но может быть несколько источников).
<pre>Выводит текст без форматирования, с сохранением пробелов и переносов текста. Может быть использован для отображения компьютерного кода, сообщения электронной почты и т.д.
<progress>Индикатор выполнения задачи любого рода.
<q>Краткая цитата.
<ruby>Контейнер для Восточно-Азиатских символов и их расшифровки.
<rb>Определяет вложенный текст как базовый компонент аннотации.
<rt>Добавляет краткую информацию сверху или снизу от символов под тегом <ruby>.
<rtc>Дополнительная аннотация.
<rp>Альтернативный текст, если браузер не поддерживает <ruby>.
<s>Не перечеркнутый, актуальный текст.
<samp>Используется для вывода текста, представляющего результат выполнения программного кода или скрипта, а также системные сообщения. Отображается моноширинным шрифтом.
<script>Сценарий действий на стороне киента (обычно в JavaScript). Указывает на текст сценария или на внешний файл сценария.
<section>Логическая область страницы (обычно заголовок).
<select>Позволяет пользователю выбрать значение из предложенного множества.
<small>Шрифт меньшего размера.
<source>Местоположение и тип встраиваемых медиаресурсов (изображение, видео, аудио).
<span>Строчные элементы (Например, выделение цветом в отдельных слов в тексте).
<strong>Акценты в тексте выделяются полужирным.
<style>Встраиваемые таблицы стилей.
<sub>Подстрочное написание символов.
<summary>Видимый заголовок для тега <details>. Отображается, как интерактивный треугольник.
<sup>Надстрочное написание символов.
<table>Таблица.
<tbody>Тело таблицы.
<td>Создает ячейку таблицы.
<template>Используется для объявления фрагментов HTML-кода, которые могут быть клонированы и вставлены в документ скриптом. Содержимое тега не является его дочерним элементом.
<textarea>Большое поле для ввода текста.
<tfoot>Нижний колонтитул страницы.
<th>Заголовок ячеек таблицы.
<time>Определяет дату/время.
<title>Заголовок документа. В браузере отображается на вкладке.
<tr>Создает строку таблицы.
<track>Субтитры для аудио и видео.
<u>Подчеркивание без дополнительного акцента.
<ul>Маркированный список.
<var>Переменные из программ, отображаются курсивом.
<video>Видеофайлы (3 формата: MP4, WebM, Ogg).
<wbr>Возможное место разрыва длинной строки.
